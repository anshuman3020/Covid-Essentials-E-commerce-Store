About Us

We have built an e-commerce website pertaining to the domain of essential items required to safeguard against the Covid-19. The main purpose of this website is for affiliate marketing, creating traffic for Amazon by our referrals. We created a user friendly interface by incorporating one-click hyperlinks wherever needed, appropriate picture gallery, the tile view of products for quick browsing. When we use e-commerce websites like Amazon, the shopping experience is not so good especially for Covid-19 essential as the specifications and technicalities and not easy to find and at times messy. Hence, a spotlight feature of our website is the clean and readable specifications column, with minimal page reloads.

We have used HTML, CSS, JS, PHP to built this website — HTML being used for structuring all the web pages and enabling linking of pictures and buttons, CSS being used for creating the layout and styling of the webpages — ensuring a clean interface and appropriate fonts sizes and colour schemes, and JavaScript being used for making the user-website engagement more interactive and making . We linked the Login and Registration pages to MySQL database using PHP, where the frontend meets the backend. MySQL is a relational database management system that uses SQL for processing data in the database. PHP is a scripting language used to building dynamic web pages. When a visitor opens a page built in PHP, the server processes the PHP commands and then sends the results to the visitor’s browser. At the time of User Login or User Registration, a request is made that is sent to the host server. PHP receives this request, obtains the desired information from the MySQL database, and then presents it on the web browser.
